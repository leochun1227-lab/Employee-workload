Employee Ticket Dashboard - Static Firebase Version

Deploy these files together:
- index.html
- app.js
- styles.css

This version does not need Node.js or a local server.
All shared data is read/written directly through Firebase:
- ctmTicketStatusMonitorV44/tempEmployeeTicketDashboard/data
- ctmTicketStatusMonitorV44/tempEmployeeTicketTable

Any customer opening the deployed website URL will see the same Firebase data.
Import Excel, mark problem tickets, undo marks, add missing tickets, and move manual tickets all persist in Firebase.

For local testing, open index.html in a browser or serve this folder with any static web server.
